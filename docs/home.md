# Tressi Documentation

Welcome to the Tressi documentation! Tressi is a high-performance, simple-to-use load testing tool designed for modern APIs.

## How to use these docs

Use the sidebar on the left to navigate through the different sections:

- **Getting Started**: Learn the basics and our core principles.
- **Usage**: Detailed guides on CLI commands and configuration.
- **Results**: Understand the math and charts behind your test results.
- **Internals**: Deep dive into how Tressi works under the hood.
- **Community**: Contribution guidelines and licensing.
