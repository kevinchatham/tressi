# Installation

Tressi is distributed as an NPM package.

## Prerequisites

- Node.js 18 or higher
- NPM or Yarn

## Global Installation

To use Tressi from anywhere on your system:

```bash
npm install -g @tressi/cli
```

## Local Installation

To use Tressi within a specific project:

```bash
npm install --save-dev @tressi/cli
```

You can then run it using `npx tressi`.
