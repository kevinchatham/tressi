# Introduction

Tressi is a modern, simple load testing tool for APIs. It is designed to be easy to use while providing powerful insights into your application's performance.

## Key Features

- **Simple Configuration**: Use a JSON file to define your tests.
- **Real-time Metrics**: Monitor your tests as they run through a web-based dashboard.
- **High Performance**: Built with Node.js worker threads and shared memory for efficient load generation.
- **Detailed Reporting**: Export results to JSON, XLSX, or Markdown.
